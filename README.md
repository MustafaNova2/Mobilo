# Description

A full-stack vehicle rental platform that allows users to explore and book vehicles online, while providing powerful management tools for admins and vendors.

---

## 📌 Core Modules

### 👤 User Features

- **Browse & Book Vehicles**  
  Users can explore available vehicles and complete bookings بسهولة.

- **Profile Management**  
  Update personal information and manage account settings.

- **Order Tracking**  
  View past and upcoming bookings, restricted to the logged-in user.

- **Account Management**  
  Supports registration, login, logout, and account deletion.

- **Email Notifications**  
  Booking confirmations are automatically sent via email with order details.

---

### 🛠️ Admin Panel

- **Booking Management**  
  View and manage all bookings including status updates.

- **Vendor Management**  
  Approve, reject, or remove vendors from the platform.

- **Vehicle Management**  
  Update or delete vehicle listings.

- **User Management**  
  Remove users and maintain platform integrity.

---

### 🚘 Vendor Features

- **Authentication System**  
  Dedicated sign-up and login flow for vendors.

- **Vehicle Listings**  
  Vendors can submit vehicles for admin approval before listing.

- **Order Notifications**  
  Get notified when users book their vehicles.

---

## ⚙️ Tech Stack

### Frontend
- React.js (Vite)
- Redux Toolkit
- Tailwind CSS
- React Hook Form
- Zod (validation)
- Google OAuth
- Razorpay (payments)

### Backend
- Node.js
- Express.js
- MongoDB
- Multer (file uploads)
- Nodemailer (emails)
- Cloudinary (media storage)
- JWT Authentication (access & refresh tokens)
- Role-Based Access Control
- MVC Architecture

### Database
- MongoDB with:
    - Aggregation pipelines
    - Referenced models
    - Optimized data structure

---

## 🚀 Key Features

- **JWT Authentication**  
  Secure login system using access and refresh tokens.

- **Role-Based Access Control**  
  Protected routes based on roles (User, Admin, Vendor).

- **Dynamic Location Selector**  
  Pickup and drop-off options adjust dynamically based on location.

- **Search, Filter & Sort**  
  Advanced vehicle browsing experience.

- **Custom UI Development**  
  Built from scratch with strong form validation using Zod and React Hook Form.

- **Google OAuth Integration**  
  Fast and secure authentication using Google accounts.

- **Email System**  
  Automated booking confirmation emails.

- **Cloudinary Integration**  
  Efficient handling of images and videos, reducing database load.

- **Optimized MongoDB Usage**  
  Structured using multiple models and references for better performance.

- **File Upload Handling**  
  Managed using Multer for vehicle images and media.

---

## 📦 Summary

This platform delivers a complete vehicle rental solution with a scalable architecture, secure authentication, and role-based workflows for users, admins, and vendors.
